Timeit: jQuery plugin for static pages to hide/display content based on date/time

Format of date: YYYY-MM-DD HH:MM

Basic examples

Hide content after a date

  • May 5th, 2014 no start, past end - hidden
  • July 10th, 2025 no start, future end - shown
  • January 10, 2014 start in the past, no end - shown
  • Dec 1, 2026 start Future, no end - hidden

    • Show content between dates

  • June 2014 - Jan 2017 whithing a range
  • June - July 2013 past range
  • Jan-Feb 2018 future range

    • Show content after a date (no expiration)

  • June 31, 2014
  • Jan 1st, 2020

    • Complex solutions

    Repetetive daily event

  • 12:10 PM - 6 PM daily

    • Repetetive event with a start/end date

  • Show content from 5AM to 12AM daily starting Sept 1, 2014


    • Processing user input errors

    When the date or time is entered with an error, .error class is added with a 'title' attribute that describes an error. This functionality is removed from minified version of the plugin.
